# Gallery Images Directory

This directory contains all the temple construction and historic photos that will be displayed in the photo gallery section.

## How to Add Images

Simply drop your images into this folder (`public/images/gallery/`) and they will automatically appear in the gallery section on the website.

### Supported Formats
- **JPG/JPEG** (Recommended)
- **PNG**
- **WebP**

### Image Naming Convention

For best results, name your files descriptively:

**Good Examples:**
```
construction-foundation-2024.jpg
temple-gopuram-progress.jpg
old-temple-1950s.jpg
sanctum-building-march-2024.jpg
priest-blessing-ceremony.jpg
```

**Avoid:**
```
IMG_1234.jpg
photo.jpg
image1.png
```

### Image Specifications

#### Recommended Dimensions
- **Minimum**: 800px width
- **Optimal**: 1200px - 1920px width
- **Aspect Ratio**: 16:9 or 4:3 (landscape preferred)

#### File Size
- **Maximum**: 2MB per image
- **Optimal**: 200KB - 500KB (use compression tools)
- **Tool Recommendation**: Use TinyPNG.com or Squoosh.app to compress

## Troubleshooting

**Images not showing?**
- Check file format (JPG, PNG, WebP only)
- Ensure images are directly in `/public/images/gallery/`
- Check file permissions
- Clear browser cache (Ctrl+Shift+R)

**Images loading slowly?**
- Compress images (aim for < 500KB each)
- Use JPG format instead of PNG
- Reduce dimensions if too large

**Need to remove an image?**
- Simply delete it from the folder
- Refresh the website
