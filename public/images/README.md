# Images Directory

This directory contains all the images used in the temple website.

## Required Images

### Deities
- **Shiva.png** - Image of Lord Vilvanadha Ishwarar (Main Deity)
- **Parvati.png** - Image of Goddess Parvati
- **Ganesha.png** - Image of Lord Ganesha
- **Kartikeya.png** - Image of Lord Kartikeya/Murugan

### Backgrounds
- **background.png** - Main background image for the website

## Image Specifications

### Deity Images
- Format: PNG or JPG
- Recommended size: 400x400 pixels (square)
- Aspect ratio: 1:1
- File size: < 200KB (optimized)

### Background Image
- Format: PNG or JPG
- Recommended size: 1920x1080 pixels
- Aspect ratio: 16:9
- File size: < 500KB (optimized)
- Should have good contrast for text overlay

## Notes

- All images should be optimized for web use
- Use descriptive alt text in the code
- Ensure images are culturally appropriate and respectful
- Consider using WebP format for better performance
